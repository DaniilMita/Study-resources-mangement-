# Study Resources Management

**Study Resources Management** este o aplicație desktop Java care ajută utilizatorii să își organizeze resursele de studiu, să genereze planuri de învățare, să urmărească progresul și să interacționeze cu un asistent AI pentru recomandări și întrebări.

---

## Functionalități

- Autentificare și creare cont.  
- Profil utilizator cu date personale și obiective de învățare.  
- Gestionare resurse de studiu (cărți, articole, note).  
- Generare planuri de studiu personalizate.  
- Monitorizare progres (materii studiate, timp alocat, note).  
- Chat AI pentru recomandări și întrebări legate de învățare.  

---

## Tehnologii

- Java 17+  
- Java Swing  
- SQLite  
- Maven  
- Jackson, SLF4J, SQLite JDBC
